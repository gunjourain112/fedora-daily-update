# My Updater (시스템 업데이트 관리자)

**My Updater**는 Linux 시스템(Fedora/GNOME 환경 최적화)을 위한 터미널 기반(TUI) 업데이트 관리 도구입니다.

단순히 `dnf update`만 실행하는 것이 아니라, `flatpak` 업데이트 및 사용자가 정의한 커스텀 스크립트들을 통합된 인터페이스에서 관리하고 실행할 수 있도록 돕습니다.

## 주요 기능

*   **통합 업데이트 메뉴**: `dnf` (시스템 패키지), `flatpak` (애플리케이션), 그리고 사용자 정의 스크립트를 한 곳에서 확인.
*   **대화형 실행 모드**: 업데이트 실행 시 터미널 제어권을 쉘에 이양하여 `sudo` 비밀번호 입력이나 패키지 매니저의 질문(`Y/n`)에 직접 응답 가능.
*   **커스텀 태스크 관리**: 설정 메뉴를 통해 나만의 업데이트 명령어나 유지보수 스크립트를 추가, 수정, 삭제 가능.
*   **직관적인 UI**: 직관적인 키보드 조작과 깔끔한 한글 인터페이스 제공.

## 설치 및 실행

### 요구 사항
*   Linux (Fedora 권장)
*   Go 1.19 이상

### 실행 방법

```bash
# 소스 코드에서 바로 실행
go run cmd/my-updater/main.go

# 빌드 후 실행
go build -o my-updater cmd/my-updater/main.go
./my-updater
```

## 사용법

### 메인 메뉴
*   **시스템 업데이트 관리**: 등록된 업데이트 태스크 목록을 확인하고 실행합니다.
*   **환경 설정**: 커스텀 업데이트 태스크를 관리합니다.
*   **종료**: 프로그램을 종료합니다.

### 업데이트 실행
1.  목록에서 방향키(↑/↓)로 원하는 작업을 선택합니다.
2.  `Enter` 키를 눌러 실행합니다.
3.  터미널 화면으로 전환되면 필요한 입력(비밀번호 등)을 수행합니다.
4.  작업이 완료되면 다시 메뉴로 돌아옵니다.

### 설정 관리
*   새로운 커스텀 명령어를 추가하거나 기존 명령어를 수정/삭제할 수 있습니다.
*   단, 기본 시스템 명령어(`dnf`, `flatpak`)는 수정할 수 없습니다.
